# بيانات التطبيق (المصحف وغيرها)

- **المسار المنطقي للمصحف:** `src/assets/data/quran.json`
- **للتشغيل:** يُخدم الملف من `public/quran.json` (ويب) ونسخته في `assets/web/quran.json` (أندرويد بعد البناء).
- لا يُستورد الملف مباشرة في الكود لتفادي بطء شاشة البداية؛ الجلب يتم عبر `fetch` للمسار المحلي فقط، ثم التخزين في IndexedDB.

لإضافة أو تحديث المصحف: ضع ملف `quran.json` في `web/public/quran.json` ثم أعد البناء ونسخ الأصول إلى الأندرويد.
